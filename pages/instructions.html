Angel's Housebrewed Coffee Instructions

Welcome to Angel's Housebrewed Coffee's training! Here you will find everything you need to be come the best Barista you can be!

Controls

The game functions very simply, with very little controls.

You have a order function, score and Number of Guests function at the top, these represent:

  • The order function represents what order you will have to make as the barista. If you correctly make the order, the order will reset, and the score function will go up. If the order is wrong however, the score function doesn't change, and a new order appears.

  • The score function represents how many orders you have correctly served out.

  • The Number of Guests representes how many guests you have served.

There are a total of 8 toppings in the Coffee Shop, these are:

  • ColdBrew

  • Cream

  • WhippedCream

  • Wafer

  • Nuts

  • Nutmeg

  • ChocaloteChip

  • Cinnamon

Happy serving!
